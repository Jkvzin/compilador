Creating a simple program:
```c
// Create declarations list
ASTNode *decls = ast_make_declarations();
ast_add_child(decls, ast_make_declaration(TYPE_INT, "x"));
ast_add_child(decls, ast_make_declaration(TYPE_FLOAT, "y"));

// Create commands list
ASTNode *cmds = ast_make_commands();
ast_add_child(cmds, ast_make_assignment("x", ast_make_num_int(42)));

// Create program
ASTNode *prog = ast_make_program("teste", decls, cmds);

// Use it...

// Clean up
ast_free(prog);
```

Creating an expression:
```c
// 2 + 3
ASTNode *expr = ast_make_binop(
    ast_make_num_int(2),
    '+',
    ast_make_num_int(3)
);
```

Creating a conditional:
```c
// if x > 0 then ... else ... fimse
ASTNode *if_stmt = ast_make_if(
    ast_make_binop(ast_make_id("x"), '>', ast_make_num_int(0)),
    cmds_then,
    cmds_else
);
```
