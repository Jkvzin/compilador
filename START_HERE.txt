================================================================================
THE LANGUAGE: SIMPLES
================================================================================

Portuguese-like pseudocode that compiles to x86-32 assembly.

Example Program:
```
programa fatorial
  inteiro n, fat, i
inicio
  leia n
  fat <- 1
  para i de 2 ate n passo 1 faca
    fat <- fat * i
  fimpara
  escreva fat
fim
```

Features:
- Variables (inteiro, flutuante)
- Arithmetic operators
- Conditionals (se/entao/senao/fimse)
- Loops (enquanto, para)
- I/O (leia, escreva)
- Logical operations
